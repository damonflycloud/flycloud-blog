Markdown概述

Markdown 是一种为网络作者而设计的文本-HTML转换的工具。Markdown允许你以一种友好的可读的方式进行网络写作然后將文本内容转换成结构化的XHTML（或者 HTML）。
因此，Markdown 就是关于两件事：

  • 一种纯文本的语法
  • 一种使用Perl语言编写的软件工具，將纯文本转化为HTML格式的文本。
